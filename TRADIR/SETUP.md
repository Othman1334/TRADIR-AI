# TRADIR AI — دليل الإعداد

## هيكلة المجلدات الصحيحة

```
TRADIR/
├── app.py
├── core/
├── templates/
└── static/
    ├── css/
    ├── js/
    ├── sounds/
    │   └── alert.mp3        ← ضع ملف الصوت هنا
    └── videos/
        └── trading-bg.mp4   ← ضع الفيديو هنا
```

## تشغيل النظام

```bash
pip install flask requests pandas numpy
python app.py
```

ثم افتح: http://127.0.0.1:5000
